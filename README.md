# Claude Skills

这个仓库用于分享和整理实用的 **Claude Skills**。

这里会持续收录适合 Claude / Claude Code 使用的技能示例，包括但不限于：

- 多智能体系统设计
- 深度研究与报告生成
- 工程开发与架构分析
- 项目管理与流程协作
- 文档生成、审查与格式化
- 其他高价值、可复用的技能模板

## 目标

这个仓库的目标是：

- 分享清晰、可复用的 Claude Skill 示例
- 提供技能设计思路、目录结构和编写参考
- 帮助开发者更高效地构建自己的 Skills
- 持续沉淀有实用价值的技能资产

## 仓库内容
